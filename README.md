# xenialStack

This provisioning setup provides a development stack using Ubuntu 16.04 Xenial LTS, with the following platforms installed:

  * nginx
  * python
  * postgresql
  * mongodb
  * node.js via nvm
  * vim 
  * git 

Additional packages:
  * libpq-dev 
  * build-essential 
  * virtualbox-guest-dkms
